# AI学术研究助手

一个专为学术研究者设计的智能化平台，基于Vue 3和TDesign Chat组件构建，集成AI对话、文献处理、知识库管理等核心功能。

## 🚀 项目特性

- **智能对话**：集成TDesign Chat组件，支持流式输出和思维链展示
- **现代化UI**：基于TDesign设计语言，响应式布局，专为桌面端优化
- **模块化架构**：清晰的项目结构，易于维护和扩展
- **TypeScript支持**：完整的类型检查和智能提示
- **专业级界面**：参考ChatGPT/Claude等顶级产品的设计标准

## 📋 功能模块

### 🎯 核心功能
- **AI对话页面**：三栏布局设计，支持对话历史管理、主聊天区域、数据报表面板
- **多模态输入**：支持文件上传、图片上传、语音输入
- **智能工具**：联网搜索、代码模式、Agent模式、翻译助手
- **深度思考**：支持思维链展示和推理过程可视化
- **学术助手**：专为学术研究场景设计的提示词和功能

### 📱 界面特性
- **响应式布局**：桌面端专用设计，合理的比例分配
- **左侧边栏**：20%屏幕宽度，支持折叠，对话历史管理
- **主聊天区**：62%屏幕宽度，充足的对话空间
- **右侧面板**：18%屏幕宽度，数据报表和统计信息
- **流畅交互**：平滑的动画过渡和响应式体验

## 🛠️ 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: TDesign Vue Next + TDesign Chat
- **构建工具**: Vite
- **路由管理**: Vue Router
- **状态管理**: Pinia
- **样式方案**: Less + CSS Variables
- **代码规范**: ESLint + Prettier

## 📐 项目结构

```
src/
├── components/          # 组件目录
│   └── layout/         # 布局组件
│       ├── AppLayout.vue    # 主布局
│       └── Sidebar.vue      # 侧边栏
├── router/             # 路由配置
├── stores/             # 状态管理
├── styles/             # 样式文件
│   ├── variables.less  # CSS变量
│   ├── base.less      # 基础样式
│   ├── layout.less    # 布局样式
│   └── components.less # 组件样式
└── views/              # 页面组件
    ├── DashboardView.vue   # 工作台
    ├── ChatView.vue        # AI对话
    ├── DocumentView.vue    # 文献处理
    ├── KnowledgeView.vue   # 知识库
    ├── GuideView.vue       # 指南
    └── SettingsView.vue    # 设置
```
